# BioLink Backend

## 專案介紹

BioLink Backend 為 BioLink 平台的後端服務，採用 Node.js 與 Express 開發，提供 API、使用者驗證、即時通訊及資料管理等功能。

---

## 功能

- 使用者登入 / 註冊 API
- JWT 身分驗證
- Firebase 資料存取
- Socket.IO 即時通訊
- AWS S3 圖片上傳
- RESTful API

---

## 技術

### Backend

- Node.js
- Express

### Database

- Firebase Firestore

### Authentication

- JWT
- bcrypt

### Cloud

- AWS S3

### Realtime

- Socket.IO
